# Desafio Front-end

### Repositórios
* [Front-End] - Repositório front-end
* [Back-End] - Repositório back-end

### Instalação
Após baixar os dois repositórios,

Executar no projeto Front-end:
```sh
$ npm install
$ ng build
```
Por fim, no projeto Back-end, executar o Spring Boot.
